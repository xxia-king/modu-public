# Contributing

感谢你关注墨读。这个项目是一个 macOS 原生 Markdown 阅读与编辑应用，欢迎提交问题反馈、功能建议和代码改进。

## 反馈问题

提交 issue 时，请尽量包含：

- macOS 版本
- 墨读版本
- 问题出现的操作步骤
- 预期结果与实际结果
- 相关截图或示例 Markdown 文件（请先移除隐私信息）

## 提交代码

1. Fork 本仓库并创建功能分支。
2. 保持改动聚焦，避免混入无关格式化或重构。
3. 提交前运行：

```bash
swift build
```

4. 如果改动影响用户可见行为，请同步更新 README 中相关说明。
5. 提交 pull request，并说明改动目的、主要实现和验证方式。

## 代码风格

- 优先沿用现有 SwiftUI 和 SwiftPM 项目结构。
- 新增 UI 应保持 macOS 原生、轻量、克制。
- 不提交构建产物、本地配置、私有工作区文件或包含隐私信息的示例文件。
